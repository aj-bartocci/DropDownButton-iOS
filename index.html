Drop Down Buttons

My own custom drop down buttons that display content from a supplied array. Originally created using Objective-C, I decided to remake them using swift. In my supplied sample project you can test out the buttons for your self. In the sample project the buttons are implemented using both Objective-C and Swift. The functionality is identical for both languages.
